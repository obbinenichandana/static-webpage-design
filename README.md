## Quick Deploy (5 Steps)

### Step 1: Create GitHub Repository
1. Go to **github.com/new**
2. Name: `codepath`
3. Select **Public**
4. Click **Create repository**

### Step 2: Upload Files
**Option A - Web Upload (Easiest)**
1. Click **Add file** → **Upload files**
2. Drag & drop **index.html** and **style.css**
3. Click **Commit changes**

**Option B - Command Line**
```bash
cd your-folder
git init
git add index.html style.css
git commit -m "CodePath: Static Webpage Design - DecodeLabs Project 1"
git remote add origin https://github.com/YOUR_USERNAME/codepath.git
git branch -M main
git push -u origin main
```

### Step 3: Enable GitHub Pages
1. Go to repo **Settings**
2. Scroll to **"Pages"**
3. Source: Select **main** branch
4. Click **Save**

### Step 4: Wait 1-2 Minutes
GitHub builds your site automatically

### Step 5: Get Your Live Link! ✨
```
https://YOUR_USERNAME.github.io/codepath/
```

**Example for Chandana:**
```
https://obbinenichandana.github.io/codepath/
```

## Questions or Issues?

### Issue: 404 Error on GitHub Pages
- Make sure files are named exactly: `index.html` and `style.css`
- Check that they're in the **main** branch root folder
- Wait 2-3 minutes for GitHub to rebuild

### Issue: Styling Not Loading
- Check that `style.css` is in the same folder as `index.html`
- Verify the link tag in HTML: `<link rel="stylesheet" href="style.css">`
- Clear browser cache (Ctrl+Shift+Delete)

### Issue: Images Not Loading
- The HTML uses external images from Unsplash
- If Unsplash is blocked, replace URLs with your own images
- Keep the same image dimensions (width/height attributes)
